##项目名称：网络聊天室

##项目描述：支持注册、群聊、私聊的网络聊天室

##实用技术：Java基础，Socket编程，多线程

##项目功能：
+ 支持注册、群聊、私聊、退出

##项目实现：
![](design.png)
+ 服务端：采用线程池调度执行服务器与客户端业务处理逻辑
   
+ 客户端：采用读写线程，分别处理交互与服务器数据连接
